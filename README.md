# React 实现的 CNode 论坛

## 涉及到的知识点
* react 
* react-router-dom
* axios
* antd

## 功能介绍
### 首页
可以通过左侧导航栏来归类每条帖子的类型。
![](https://tva1.sinaimg.cn/large/0081Kckwgy1gm8ba7bclhj31k00u07fd.jpg)

点击每条帖子可以进入文章详情页面。
![](https://tva1.sinaimg.cn/large/0081Kckwgy1gm8befsiv2j31k90u0dq1.jpg)

点击每条帖子左侧头像可以进入用户详情页面。
